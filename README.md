# TXTILPROS Employee Reporting Frontend

## Run

1. Copy `.env.example` if you need a custom API URL.
2. Start the backend on `http://localhost:5000`.
3. Run `npm install` and `npm run dev`.

## Features

- Secure login with employee and admin roles
- Mobile-friendly daily work reporting
- Photo uploads with preview
- Weekly report views
- Admin employee management
- Google Sheets sync status display
